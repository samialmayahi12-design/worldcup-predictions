# 🏆 دوري توقعات كأس العالم 2026

تطبيق ويب مستقل لتوقّع نتائج مباريات المونديال والتنافس على لوحة النقاط — جاهز للنشر مجانًا على **Vercel** مع قاعدة بيانات **Supabase**.

- 104 مباراة كاملة من جدول البطولة، بأعلام الدول.
- تسجيل بالاسم + الرقم الوظيفي + رمز سري.
- نظام نقاط: نتيجة دقيقة = 5، اتجاه صحيح = 2، مع مضاعفة في الأدوار الإقصائية.
- صاحب الرقم الوظيفي **190** هو المدير: يملك لوحة تحكم ويعيّن مدخلي النتائج.
- لوحة متصدرين مشتركة، وجلب آلي اختياري للنتائج.

---

## الخطوات (تقريبًا 15 دقيقة، كلها مجانية)

### 1) أنشئ قاعدة بيانات Supabase
1. ادخل [supabase.com](https://supabase.com) وسجّل دخولك (مجاني)، ثم **New Project**.
2. اختر اسمًا وكلمة مرور لقاعدة البيانات وانتظر تجهيزها.
3. من القائمة الجانبية افتح **SQL Editor** ← **New query**.
4. انسخ محتوى ملف `supabase-schema.sql` كاملًا، الصقه، واضغط **Run**. سيُنشئ الجداول.
5. افتح **Project Settings ← API** وانسخ:
   - **Project URL** (مثل `https://abcd.supabase.co`)
   - مفتاح **service_role** (السري — تحت "Project API keys").

### 2) ارفع المشروع إلى GitHub
1. أنشئ مستودعًا جديدًا (private) على [github.com](https://github.com).
2. ارفع كل ملفات هذا المجلد إليه (عبر واجهة GitHub أو الأوامر):
   ```bash
   git init
   git add .
   git commit -m "World Cup 2026 predictions app"
   git branch -M main
   git remote add origin https://github.com/<حسابك>/<المستودع>.git
   git push -u origin main
   ```

### 3) انشر على Vercel
1. ادخل [vercel.com](https://vercel.com) وسجّل عبر حساب GitHub (مجاني).
2. **Add New ← Project** واختر المستودع. سيكتشف Vercel أنه Next.js تلقائيًا.
3. قبل الضغط على **Deploy**، افتح **Environment Variables** وأضف:

   | المتغير | القيمة |
   |---|---|
   | `SUPABASE_URL` | رابط مشروع Supabase |
   | `SUPABASE_SERVICE_ROLE_KEY` | مفتاح service_role السري |
   | `AUTH_SECRET` | أي نص عشوائي طويل (مثلًا من [randomkeygen.com](https://randomkeygen.com)) |
   | `ANTHROPIC_API_KEY` | (اختياري) لتفعيل الجلب الآلي — اتركه فارغًا للإدخال اليدوي |

4. اضغط **Deploy**. بعد دقيقة ستحصل على رابط مثل `https://your-app.vercel.app`.

### 4) عيّن نفسك مديرًا
- سجّل حسابًا في التطبيق برقمك الوظيفي **190** ← سيظهر لك تبويب **«لوحة التحكم ⭐»** ومنه تمنح زملاءك صلاحية إدخال النتائج.

أرسل رابط Vercel لزملائك، وكل واحد يسجّل بنفسه. 🎉

## تشغيل محلي (اختياري للتجربة قبل النشر)
```bash
npm install
cp .env.example .env.local   # ثم عبّئ المتغيرات
npm run dev                  # http://localhost:3000
```

## ملاحظات
- **التكلفة:** خطط Vercel وSupabase المجانية تكفي تمامًا لعشرات أو مئات المستخدمين.
- **الأمان:** مفتاح `service_role` يُستخدم على الخادم فقط ولا يصل للمتصفح أبدًا. المصادقة بسيطة ومناسبة لمسابقة داخلية بين زملاء.
- **الجلب الآلي للنتائج:** يتطلب `ANTHROPIC_API_KEY` (مدفوع حسب الاستخدام، تكلفته زهيدة جدًا لهذا الغرض). بدونه يعمل كل شيء بالإدخال اليدوي من مدخلي النتائج.
- **قفل التوقعات:** يُمنع تعديل التوقع بعد انطلاق المباراة (يُفرض على الخادم، لا يمكن تجاوزه من المتصفح).
